# 🤖 WhatsApp AI Sales Bot (Starter Template)

A lightweight, open-source Node.js template to connect OpenAI's GPT models directly to WhatsApp. This starter kit allows you to deploy a basic conversational agent for customer support, lead qualification, or FAQ handling.

## ⚙️ Features
* Respond to incoming WhatsApp messages using OpenAI (GPT-3.5/GPT-4).
* Maintain basic conversational context.
* Easy configuration using `.env` variables.

## 🛠️ Quick Setup

1. **Clone the repository:**
   `git clone https://github.com/your-username/whatsapp-ai-salesbot-starter.git`
2. **Install dependencies:**
   `npm install`
3. **Configure Environment:**
   Rename `.env.example` to `.env` and add your API keys.
4. **Run the bot:**
   `node index.js`
